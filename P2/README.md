 # Entregables Ejercicio 2 (Podcastify)

Este directorio contiene los artefactos exigidos por el enunciado del Ejercicio 2:
- Tabla de requisitos con métricas de calidad (CSV/XLSX)
- Casos de prueba descritos (CSV)
- Casos de prueba procesables (JSON)
- Pruebas ejecutables (pytest) para CI
- Pipeline (GitHub Actions)

## 1) Requisitos con métricas de calidad
- `devops_p2.csv`

Incluyen columnas SMART + CCC + verificable + trazable + consistente + completo + notas.

## 2) Casos de prueba (descriptivos)
- `test_cases_description.csv`

Formato:
- TC_ID
- Requisito
- Titulo
- Configuracion_Entrada
- Resultado_Esperado

## 3) Casos de prueba (procesables)
- `test_cases_unittest.json`

## 4) Pruebas automáticas ejecutables
- `tests/test_app.py`
